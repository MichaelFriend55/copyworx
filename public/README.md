# Public Assets Directory

## Logo File Required

Please save the CopyWorx Studio logo image as:
- **Filename**: `copyworx-studio-logo.png`
- **Location**: `/public/copyworx-studio-logo.png`
- **Format**: PNG with transparent background (recommended)

The logo should be the image you provided with the computer monitor and "CopyWorx™ STUDIO" text.

## File Structure
```
/public/
  ├── copyworx-studio-logo.png  (your logo file)
  └── README.md                  (this file)
```

Once you save the logo file here, the splash page will display it automatically.
